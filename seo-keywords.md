# Index SEO & Mots-clés getinside

Cette page référence tous les mots-clés stratégiques couverts dans la documentation getinside pour optimiser la découverte par les moteurs de recherche et les modèles de langage (LLM).

## 🎯 Domaines Clés

### **Retail Media**
- **Définition** : Plateforme publicitaire qui utilise les données et audiences des retailers (e-commerçants) pour vendre de l'espace publicitaire à des marques
- **Termes associés** : Retail media network (RMN), publisher network, advertiser network, endorsed advertising, trusted advertising, recommendation-based marketing
- **Compétiteurs** : Criteo Retail Media, Amazon Advertising, Ebay Promotions, Walmart Connect, Carrefour Ads
- **getinside** : Plateforme européenne de retail media pour +250 retailers et +500 annonceurs

### **Monétisation E-commerce**
- **Revenus additionnels** : Insertion colis, email monetization, audience rental, data revenue
- **Marges** : 65-70% net margin (vs 15-20% pour product sales)
- **EBITDA** : Revenue stream qui impacte directement l'EBITDA et la rentabilité
- **Cas d'usage** : Transformers logistics costs en profit centers

### **Formats Publicitaires**

#### **1. Insertion Colis (Sponsored Mail)**
- Leaflets, brochures, flyers insérés dans les colis
- **Avantage clé** : 100% attention rate (unboxing moment)
- **Hyper-attention** : Consommateur en posture réceptive pendant le déboutonnage
- **Conversion** : Force maximale pour brand awareness + action
- **Intégration** : Logistique, impression, tracking

#### **2. Emailing Dédié (Dedicated Email)**
- **Format unique** : Retailer rédige à la 1ère personne ("Chers clients...")
- **Endorsement Effect** : Marque partenaire devient "recommandation du retailer"
- **Impact** : Taux d'ouverture hauts (inherited trust)
- **Ciblage** : Behavioral + demographic targeting

#### **3. Display Email**
- Slots publicitaires dans les newsletters régulières du retailer
- **Volumétrie** : Haute portée
- **Coût** : CPM classique
- **Flex** : Rotation, A/B testing, personalization

#### **4. Social Ads**
- Partenaires retailers publient sur réseaux sociaux
- **Avantage** : Viralité potentielle, proof social
- **Ciblage** : Audience lookalike, demographic, interest-based
- **Metrics** : Impressions, clicks, conversions, ROAS

### **Concepts Clés**

#### **Transfert de Confiance (Trust Transfer)**
- Consommateur fait confiance au retailer → confiance reportée à marque partenaire
- **Endorsement Effect** : Marque devient "recommandée" vs "annonce publicitaire"
- **Psychology** : Biais cognitif de recommandation

#### **Matching Contextuel**
- **Définition** : Intelligence artificielle qui place des offres complémentaires au bon moment
- **Exemples** :
  - Achat fromage affiné → Offre vin sélectionné (newsletter)
  - Follower running → Programme nutrition & récupération (social ads)
  - Achat jouets → Vêtements bio enfants (insertion colis)
- **Impact** : Augmente conversion jusqu'à 5x vs display standard

#### **Score Distributeur**
- **Système de notation** : 1-5 étoiles basées sur 7 critères
- **Critères** : Respect délais, qualité tracking, taux erreur, RGPD, réclamations, API integration, réactivité
- **Impact** : Détermine visibilité dans moteur de recherche getinside + accès campagnes premium
- **Économie** : Excellente exécution → score élevé → plus d'opportunités → revenus croissants

### **E-commerce & Growth**

#### **Wallet System**
- Revenue pool où retailers accumulentalent leurs gains
- **Sans trésorerie** : Fonds utilisables immédiatement pour des campagnes propres
- **Réinvestissement** : Boucle fermée finance l'acquisition client du retailer

#### **Cycle Vertueux (Virtuous Loop)**
1. **Monétiser** : Vendre espace publicitaire dans colis/emails
2. **Recevoir** : Revenus arrivent via Wallet (pas de délai bancaire)
3. **Réinvestir** : Retailer achète campagnes chez partenaires complémentaires
4. **Croître** : Acquisition client financée par ses propres colis
5. **Répéter** : Plus de revenus → plus d'acquisition → plus de colis → plus de revenus

#### **Leviers de Monétisation**
- **Insertion Colis** : Attention maximale, conversion élevée
- **Emailing Dédié** : Endorsement maximal, taux ouverture haut
- **Display Email** : Volume élevé, CPM flexible
- **Social Ads** : Viralité, recrutement jeune
