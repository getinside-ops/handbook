# Résolution de Problèmes — Annonceurs

::: details Ma campagne n'est pas encore en ligne — que vérifier ?

Plusieurs causes possibles :

1. **Validation en attente** — Connectez-vous sur [app.getinside.media](https://app.getinside.media/) et vérifiez l'onglet **"Mes Campagnes"**. Un bouton "Valider la campagne" actif signifie qu'une action de votre part est requise.

2. **Créatif non soumis** — Pour le Display Email ou l'Emailing Dédié, les éléments créatifs doivent être transmis au distributeur via la plateforme. Vérifiez que votre visuel et votre URL de destination ont bien été envoyés.

3. **Délai d'impression** — Pour l'Insertion Colis, comptez 2 à 4 semaines entre la validation du BAT et la réception des flyers chez le distributeur.

Si aucune de ces causes ne s'applique, contactez [support@getinside.fr](mailto:support@getinside.fr).

:::

::: details Mon créatif a été refusé — quelles sont les raisons fréquentes ?

Les refus les plus courants :

- **Résolution insuffisante** — Le fichier print doit être en 300 DPI minimum, mode CMJN. Vérifiez les [spécifications techniques](/advertisers/sponsored-mail/design-guide/print-specs).
- **Fond perdu absent** — Pour l'Insertion Colis, 3 mm de fond perdu sur chaque côté sont obligatoires.
- **Mentions légales manquantes** — Loi AGEC (logos recyclage), mentions d'offre (durée de validité, conditions). Consultez le [guide des mentions légales](/advertisers/sponsored-mail/design-guide/legal).
- **Bannière hors format** — Pour le Display Email, vérifiez les [spécifications acceptées](/advertisers/display-email/specs).
- **Contenu non conforme** — Offre sans date de validité, promesse trompeuse ou contenu incompatible avec la charte du distributeur.

:::

::: details Je ne trouve pas le bouton de validation BAT

Le bouton apparaît uniquement quand un BAT est en attente de votre validation :

1. Connectez-vous sur [app.getinside.media](https://app.getinside.media/)
2. Allez dans **Mes Campagnes**
3. Sélectionnez la campagne concernée
4. Cherchez le bouton **"Valider la campagne"** dans l'onglet de suivi

Si le bouton n'apparaît pas, le distributeur n'a pas encore soumis le BAT. Contactez [support@getinside.fr](mailto:support@getinside.fr) pour vérifier l'état.

:::

::: details Ma campagne a été mise en pause — pourquoi ?

Causes possibles :

- **Stock épuisé** (Insertion Colis) — Le distributeur a distribué tous les flyers reçus. Contactez le support pour un éventuel réapprovisionnement.
- **Demande de correction** — Le distributeur a signalé un problème avec votre créatif en cours de diffusion. Vérifiez vos notifications sur la plateforme.
- **Fin de période** — La date de fin de campagne a été atteinte.
- **Problème technique** — Contactez [support@getinside.fr](mailto:support@getinside.fr) si aucune cause ci-dessus ne s'applique.

:::

::: details Je veux modifier une campagne déjà en cours

Les modifications possibles dépendent du format :

- **URL de destination** — Modifiable si vous utilisez une URL dynamique. Contactez votre chargé de compte.
- **Visuel ou créatif** — Non modifiable après validation du BAT pour l'Insertion Colis (le flyer est imprimé). Pour les formats email, une modification est parfois possible avant l'envoi — contactez le support.
- **Budget ou volume** — Contactez [support@getinside.fr](mailto:support@getinside.fr) — un avenant contractuel peut être nécessaire.

:::

::: details Comment contacter le support ?

- **Support général :** [support@getinside.fr](mailto:support@getinside.fr) — délai de réponse : 24 à 48h ouvrées
- **Studio créatif :** [studio@getinside.fr](mailto:studio@getinside.fr) — production graphique, BAT, création
- **Opérations :** [benoit@getinside.fr](mailto:benoit@getinside.fr) — questions contractuelles ou urgences

Précisez votre numéro de campagne dans l'objet de votre email pour accélérer le traitement.

:::
