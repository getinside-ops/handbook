# Compte, Équipe & Audiences

::: details Comment ajouter un nouvel utilisateur à mon compte ?
Rendez-vous dans **Mon Organisation → Équipe** et cliquez sur **+ Ajouter un membre**. Renseignez l'adresse email et le rôle. Recommandation : ayez toujours au moins un membre de secours pour garantir une réactivité sous 48h aux propositions de campagne.
:::

::: details Comment référencer une nouvelle audience ?
Allez dans **Monétisation → Catalogue** et cliquez sur **+ Nouvelle audience**. Remplissez la fiche avec soin : marque, persona, panier moyen. Ces informations sont ce que les annonceurs consultent pour sélectionner vos audiences.

→ [Guide détaillé : Configuration d'Audience](/publishers/audience-setup)
:::

::: details Comment rendre ma fiche audience attractive pour les annonceurs ?
Les annonceurs parcourent le catalogue comme une place de marché. Pour maximiser vos chances de recevoir des offres :

- Ajoutez des visuels : photos de colis, captures d'emails, exemples d'emballages.
- Décrivez précisément votre persona : âge, catégorie produit, fréquence d'achat, zone géographique.
- Indiquez un panier moyen (AOV) réaliste — un chiffre précis est plus convaincant qu'une fourchette.
- Activez plusieurs formats : une audience proposant Insertion Colis + Emailing est plus attractive qu'une audience mono-format.

→ [Guide complet : Configuration d'Audience](/publishers/audience-setup)
:::

::: details Faut-il créer plusieurs comptes si j'ai plusieurs sites ?
Non. Un seul compte peut héberger autant de fiches audiences que vous avez de sites ou de flux logistiques distincts. Chaque audience est indépendante : tarifs, formats disponibles et périodes d'ouverture sont configurés séparément.
:::

::: details getinside opère-t-il à l'international ?
Oui, getinside est présent dans toute l'Europe. Vous pouvez ajouter des audiences européennes et recevoir des campagnes ciblées pour ces marchés depuis le même compte.
:::

::: details Comment suivre mes commissions d'affiliation ?
L'état de vos filleuls est visible dans l'onglet **Affiliation**. Vous percevez une commission récurrente de **20%** sur les frais de plateforme générés par leur activité. Vos gains apparaissent dans **Transactions → Revenus** et peuvent être retirés ou réinvestis depuis votre Wallet.
:::

::: details Pourquoi mon Score Distributeur a-t-il baissé ?
Le score est recalculé dynamiquement. Causes fréquentes : délai de réponse supérieur à 48h, refus de campagnes sans contre-proposition, volumes distribués inférieurs aux engagements, CPM trop élevé par rapport à la concurrence.

→ [Guide complet : Score Distributeur](/publishers/score-distributor)
:::
