# Configuration de l'Audience

Votre fiche audience est ce que les annonceurs voient en premier quand ils vous trouvent sur la plateforme. Une fiche complète et précise améliore directement votre Score Distributeur et votre visibilité dans l'algorithme de Smart Matching.

## Comment ça marche

<div class="gi-step">
  <div class="gi-step-num">1</div>
  <div class="gi-step-body">
    <h3>Renseignez l'identité de votre audience</h3>
    <p>Présentez votre site, vos valeurs et votre positionnement. Dans le champ <strong>Persona</strong>, soyez descriptif : <em>"Jardinier amateur ou expert cherchant des produits durables pour entretenir son potager."</em> Uploadez un logo et une bannière qui représentent votre univers. Cochez les engagements RSE uniquement si vous pouvez les justifier.</p>
  </div>
</div>

<div class="gi-step">
  <div class="gi-step-num">2</div>
  <div class="gi-step-body">
    <h3>Complétez les données démographiques</h3>
    <p>Ces champs alimentent directement l'algorithme de Smart Matching. Une fiche incomplète réduit mécaniquement votre visibilité dans les résultats de matching.</p>
  </div>
</div>

<div class="gi-step">

  <div class="gi-step-body">

  </div>
</div>

<div class="gi-step">
  <div class="gi-step-num done">4</div>
  <div class="gi-step-body">

:::

## Gérer plusieurs audiences et sites

Un seul compte suffit, même si vous opérez plusieurs sites e-commerce. Chaque site peut avoir sa propre fiche audience avec ses données spécifiques.

**Espace :** Monétisation → Catalogue → + Nouvelle audience

Bonnes pratiques pour un catalogue multi-audiences :
- Nommez clairement chaque audience (ex: "Site Mode Femme", "Site Jardin B2C").
- Renseignez des données distinctes par site — ne mutualisez pas les chiffres.
- Activez uniquement les formats que vous pouvez réellement honorer sur ce site.

getinside opère dans toute l'Europe. Ajoutez vos audiences européennes pour recevoir des campagnes ciblées par pays.

## Questions fréquentes

**Comment ajouter un utilisateur à mon compte ?**
Mon Organisation → Équipe → + Ajouter un membre. Recommandé : ayez toujours un backup pour garantir la réactivité sous 48h.

**Puis-je gérer plusieurs sites depuis un seul compte ?**
Oui. Un compte getinside peut héberger autant d'audiences que vous le souhaitez — une par site ou par segment client.

**Où suivre mes filleuls ?**
Dans l'onglet Affiliation. Vous touchez 20% de commission récurrente sur les frais de plateforme de vos filleuls. → [Programme d'Affiliation](/publishers/affiliation)
