## Formats de bannières

| Format | Dimensions | Emplacement typique |
| :--- | :--- | :--- |
| **Leaderboard** (Bannière Large) | 600 × 150 px | Header ou cœur du mail |
| **Pavé** (Rectangle) | 300 × 250 px | Colonne latérale ou pied de mail |

:::tip Fond blanc — ajoutez une bordure
Si votre bannière a un fond blanc, ajoutez une fine bordure grise de 1 px (`#cccccc`) pour qu'elle ne se fonde pas dans l'arrière-plan de l'email.
:::
