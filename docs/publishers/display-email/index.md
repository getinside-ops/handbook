# Encart Newsletter (Display Email)

Insérez une bannière partenaire dans des emails que vous envoyez déjà. La création vient de l'annonceur, vous ne produisez rien, et chaque envoi devient une source de revenu.

::: tip Côté annonceur
Vous êtes une marque et souhaitez diffuser une bannière dans les newsletters du réseau ? Consultez le [guide Encart Newsletter pour les Annonceurs](/advertisers/display-email/).
:::

## Le principe

Vos newsletters et emails transactionnels (confirmation de commande, suivi d'expédition, relance panier) partent par milliers chaque semaine. Le Display Email ajoute un emplacement publicitaire dans ces emails : l'annonceur fournit la bannière, getinside vous transmet le bloc à intégrer, vous le placez dans votre routeur. C'est tout.

Vous restez maître de l'éditorial : vous choisissez quels annonceurs apparaissent, et un annonceur concurrent de votre activité ne passera jamais sans votre accord.

## Côté technique

L'intégration est légère : un bloc HTML ou une image à insérer dans le gabarit de votre routeur (Klaviyo, Brevo, Mailchimp, Splio, etc.). getinside fournit le visuel et le lien de tracking. Une fois l'emplacement créé, les campagnes suivantes réutilisent le même slot.

## Bonnes pratiques

- **Placez l'encart en cohérence avec votre contenu** — sous le corps principal de la newsletter, ou en pied d'email transactionnel, là où il ne gêne pas le message client.
- **Gardez une fréquence raisonnable** — un encart par email suffit ; l'objectif est un revenu régulier, pas une saturation.
- **Déclarez vos envois** pour déclencher la facturation (voir le guide d'intégration).
