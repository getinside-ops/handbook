# Parcel Insert (Sponsored Mail)

Slip your offer (flyer, card, sample) inside the parcels of carefully selected e-retailers — and reach your customers at the moment of unboxing.

## Targeting & Audiences

Select your distributors based on four criteria:

| Criterion | Available options |
| :--- | :--- |
| **Product universe** | Beauty, Food, Tech, Home, Sport, Fashion, Wellness... |
| **Average Order Value (AOV)** | High-end, Mass Market, Discounter, Budget |
| **Buyer profile** | B2C (individuals) or B2B (professionals) |
| **Geography** | France, Belgium, Spain, Germany... |

:::info Guaranteed sector exclusivity
Your offer is always inserted with a **non-competing** e-retailer. And in the same parcel, two competing offers can never coexist.
:::

## Performance & ROI

Benchmarks observed on the getinside network:

| Indicator | Network benchmark |
| :--- | :--- |
| **Scan rate (CTR)** | 1% to 10% — **network average: 2.87%** |
| **Post-scan conversion rate (CVR)** | 10% to 30% |

::: info Average CTR vs. traditional digital
**2.87%** average CTR on the getinside network — compared to **0.90%** for Meta Ads. Parcel inserts generate up to **3× more qualified clicks** thanks to the hyper-attention of the unboxing moment.
:::

:::tip Maximize ROI
- **Monetary value**: *"10€ off"* converts much better than *"-10%"*. Concrete value is immediately perceived.
- **QR Code + UTM**: combine the getinside QR code with UTM parameters (`utm_source=getinside&utm_medium=parcel-insert`) to track sales in Google Analytics.
- **Unique promo code**: add a specific code per distributor for offline attribution.
- **Validity period**: allow at least 3 months — recipients often keep flyers for several weeks before scanning them.
:::
