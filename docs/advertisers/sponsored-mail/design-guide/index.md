# Guide de Conception

Le déballage d'un colis est un moment à part. Le consommateur vient de recevoir quelque chose qu'il attendait — il est attentif, les mains occupées, l'esprit ouvert. Votre flyer a moins de 3 secondes pour en tirer parti.

Ce guide couvre tout : stratégie d'offre, mécanique, copywriting, mise en page, et validation technique.

:::warning Validation obligatoire
Aucun contenu n'est diffusé à un distributeur sans validation préalable de getinside via la plateforme. Soumettez votre BAT numérique avant toute impression.
:::

## L'Offre

L'offre est la décision la plus importante de votre flyer. Une bonne offre compense une mise en page moyenne. L'inverse est rarement vrai.

### Valeur monétaire vs pourcentage

:::tip "10€ offerts" convertit mieux que "-10%"
Une valeur concrète est perçue immédiatement. Un produit offert fonctionne encore mieux — le levier psychologique du cadeau surpasse celui de l'économie.
:::

### Durée de validité

:::info Minimum 3 mois
Les destinataires gardent souvent un flyer plusieurs semaines avant de le scanner. Une offre qui expire trop vite est une conversion perdue.

Prévoyez au minimum 3 mois de validité à partir de la date d'impression.
:::

### Code promo

Un code promo renforce le sentiment d'exclusivité et permet de tracer les conversions offline :

- Un code **unique par distributeur** — ex : "GREENWEEZ20", "BERCEAU15"
- Court et mémorisable, sans caractères spéciaux (pas de `_`, `-`, `;`)
- Inclure le nom ou l'univers du distributeur renforce la cohérence et le taux d'utilisation

### L'alignement audience

Avant de valider votre mécanique, posez-vous une question : **"Pourquoi ce contenu est-il destiné à cette audience ?"**

Si la réponse n'est pas évidente pour vous, elle ne le sera pas pour le destinataire. La cohérence entre l'univers du distributeur et votre offre est ce qui fait la différence entre un flyer conservé et un flyer jeté.

## Le Copywriting

### L'accroche

Ne commencez pas par le nom de votre marque. Commencez par le bénéfice client.

- **À éviter :** "Découvrez la nouvelle collection [Marque]"
- **À préférer :** "Votre routine beauté offerte dans ce colis" / "Dites adieu à [Problème] dès aujourd'hui"

### La promesse

L'offre doit être l'élément le plus lisible du flyer. Utilisez des verbes d'action : "Profitez", "Recevez", "Économisez".

### L'urgence

Donnez une raison d'agir maintenant :

- "Offre réservée aux clients de [Nom du Partenaire]"
- "Valable pour les 500 prochaines commandes"

## L'UX papier

### Hiérarchie visuelle

L'œil suit un parcours en Z sur un format A5/A6 :

- **Haut gauche** — Logo du distributeur (endossement, rassurance)
- **Centre** — L'offre et le visuel produit
- **Bas droite** — Le QR code et le CTA, là où le pouce tient naturellement le flyer

### QR code

Le QR code est le seul pont vers la conversion.

- Fond blanc, module noir — évitez les fonds texturés ou clairs
- Laissez de l'espace autour du code pour faciliter la mise au point de l'appareil photo
- Ajoutez toujours une instruction courte à côté : "Scannez pour profiter de l'offre"
- **Taille minimum : 25 × 25 mm** (voir [spécifications techniques](./print-specs))

Un CTA explicite ("Scannez pour obtenir votre réduction de 10€") peut générer jusqu'à 10 fois plus de scans qu'un CTA vague ("En savoir plus").

### Lisibilité

- Corps de texte : 12pt minimum
- Mentions légales : 8pt minimum
- Texte clair sur fond clair : à éviter

## Après le scan

L'expérience post-scan se fait à 99% sur smartphone. Votre landing page doit être prête avant l'impression.

:::info Trois points non négociables
1. **Responsive** — la page s'adapte aux petits écrans
2. **Rapide** — chargement immédiat même en 3G/4G
3. **Simple** — un seul formulaire court, un seul appel à l'action
:::

## Questions fréquentes

::: details Un distributeur refuse mon visuel — que faire ?
Le distributeur a un droit de veto sur la **marque** (concurrent direct, catégorie incompatible avec son image), mais pas sur la **direction artistique**.

Si votre fichier respecte les spécifications techniques — fond perdu 2 mm, zone tranquille 3 mm, résolution 300 DPI, mentions légales présentes — il est conforme. Le distributeur ne peut pas le rejeter pour des raisons subjectives de goût ou de style.

Marche à suivre :
1. Vérifiez que votre fichier respecte les spécifications d'impression.
2. Si tout est conforme, transmettez votre BAT à [studio@getinside.fr](mailto:studio@getinside.fr) pour validation officielle.

La validation getinside prime sur l'appréciation subjective du distributeur.
:::

::: details Comment suivre les performances de ma campagne ?
Vous disposez d'un double suivi dans votre espace annonceur getinside :
- **Performance** — scans de vos QR codes en temps réel
- **Distribution** — volumes de flyers insérés, mis à jour chaque début de semaine par le distributeur

Pour suivre les ventes dans Google Analytics, ajoutez des paramètres UTM à votre URL de destination. Ajoutez aussi un code promo unique par distributeur pour l'attribution offline.
:::
