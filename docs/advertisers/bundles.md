# Bundles & Media Kits

Un Bundle, c'est un dispositif scénarisé qui active plusieurs inventaires d'un même e-commerçant en une seule opération : insertion colis, email, social, display. Là où une campagne isolée génère une exposition, un Bundle crée un effet de présence qui multiplie les taux de conversion.

## Comment ça marche

  <div class="gi-step-num">1</div>
  <div class="gi-step-body">

<div class="gi-step">

  <div class="gi-step-body">

    <p>Via la plateforme : visuels (logo HD, packshots), offre ou dotation, code promo de consolation. Le distributeur crée la landing page co-brandée sur son domaine.</p>

## Performances

<div class="gi-value-grid">
  <div class="gi-value-card">
    <strong>Jusqu'à 20% de CTR</strong>
    <p>Les formats gamifiés (jeu-concours, roue de la fortune, quiz) atteignent jusqu'à 20% de taux de clic — contre 2,64% en moyenne réseau pour l'insertion colis standard.</p>
  </div>
  <div class="gi-value-card">
    <strong>Leads opt-in qualifiés</strong>
    <p>Chaque participant consentant enrichit votre CRM avec un contact qualifié par les habitudes d'achat du distributeur. Aucun perdant : tous reçoivent un code promo de consolation.</p>
  </div>
  <div class="gi-value-card">
    <strong>Mémorisation renforcée</strong>
    <p>Trois expositions (physique + email + social) sur le même message créent un effet de présence qui améliore mécaniquement la mémorisation et la conversion.</p>
  </div>
</div>

::: tip Exemple terrain
AJY (marque bien-être) a obtenu 21,6% de taux de scan avec un quiz "Découvrez votre Dosha" — l'un des meilleurs scores observés sur le réseau.
:::

## Mécaniques disponibles

| Mécanique | Objectif principal |
| :--- | :--- |
| **Promotion** | Conversion directe via code promo ou bon de réduction |
| **Jeu-Concours** | Leads opt-in + code promo de consolation pour tous |
| **Contenu / Brand Content** | Notoriété et positionnement expert |
| **Drive to Store** | Trafic en point de vente physique |
| **Drive to App** | Installs qualifiées |
| **Feedback / Notation** | Collecte d'avis et panels consommateurs |

## Tarification

| Poste | Coût |
| :--- | :--- |
| **Budget média global** | Forfait — entre 10 000€ et 15 000€ pour un dispositif complet sur un mois |
| **Frais de plateforme** | 12% du budget média investi |
| **Studio Créatif** (optionnel) | 550€ HT — production Email, Social, Print (hors Landing Page) |
| **Dotation** | À votre charge — valeur marchande + expédition aux gagnants |

Le prix au forfait est généralement plus avantageux que la somme des leviers achetés séparément. Vous pouvez ajuster le contenu du pack (retirer un levier, modifier le volume) en discussion directe avec le retailer.
