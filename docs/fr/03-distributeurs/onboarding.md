# Onboarding et Fonctionnement
{: .fs-9 }

Bienvenue sur getinside ! Ne vous contentez pas d'attendre les campagnes : prenez le contrôle de votre régie, activez vos partenaires historiques et maximisez vos revenus.
{: .fs-6 .fw-300 }

<hr class="my-6">

<h2 class="text-purple-200 mb-6">Phase 1 : Configuration et Activation</h2>

{% capture step1 %}
Tout commence par un call de démo avec l'équipe getinside.
- L'équipe crée votre compte e-commerçant.
- Tous les contacts fournis reçoivent un email d'invitation.
- **Action :** Cliquez sur le lien dans l'email pour créer votre mot de passe et accéder à la plateforme.
{% endcapture %}
{% include step.html index="1" title="Démo & Création du compte" content=step1 %}

{% capture step2 %}
Avant de créer vos audiences, définissez vos contraintes logistiques (indispensable pour l'algorithme).
{% include callout.html type="info" content="**📍 Adresse de l'entrepôt :** Lieu de livraison des supports.<br>**⏱️ Délai d'ingestion (Lead Time) :** Délai minimum entre la livraison et le début de la diffusion (automatiquement pris en compte par la plateforme)." %}
{% endcapture %}
{% include step.html index="2" title="Configuration Logistique" content=step2 %}

{% capture step3 %}
Dans l'onglet **Monétisation**, l'équipe e-commerçant crée ses audiences.

[Voir le guide détaillé de la Fiche Audience & Score →](audience-setup){: .btn .btn-outline .btn-sm .mt-2 }
{% endcapture %}
{% include step.html index="3" title="Création des Audiences" content=step3 %}

{% capture step4 %}
Pour chaque audience créée, ajoutez les types de campagnes acceptés et leurs critères :
- **📦 Asile Colis :** Capacité (max flyers simultanés), formats acceptés, volumes prévisionnels par semaine sur 12 mois.
- **📧 Digital :** CPM indicatif, exemples de campagnes passées, infos de performance (Taux d'ouverture, Clics).
{% endcapture %}
{% include step.html index="4" title="Ajout des Types de Campagnes" content=step4 %}

{% capture step5 %}
Dès que vous estimez avoir complété les infos, avertissez l'équipe getinside.
- Nous proposons un call pour confirmer que toutes les infos nécessaires sont remplies.
- Rappel des aspects techniques et des bonnes pratiques.
{% endcapture %}
{% include step.html index="5" title="Validation (Call de contrôle)" content=step5 %}

{% capture step6 %}
Suite au call :
- ✅ Les audiences et types de campagnes validés sont **activés**.
- ⏸️ Ceux qui sont incomplets restent non-référencés en attendant.
{% endcapture %}
{% include step.html index="✓" color="green" title="Activation" content=step6 %}


<hr class="my-6">

<h2 class="text-purple-200 mb-6">Phase 2 : Gestion des Campagnes</h2>

{% capture step7 %}
Une fois activées, les annonceurs peuvent voir vos audiences sur la plateforme et commencer à vous proposer des campagnes.
{% endcapture %}
{% include step.html index="7" title="Visibilité Marketplace" content=step7 %}

{% capture step8 %}
Lorsqu'une campagne est proposée, tous vos utilisateurs reçoivent une notification.

{% include callout.html title="Ne refusez pas, négociez !" content="L'objectif est de remplir votre inventaire. Plutôt que de refuser une offre :<br>• **💰 Prix trop bas ?** Faites une contre-proposition tarifaire.<br>• **📅 Dates indisponibles ?** Proposez une autre période de diffusion.<br>• **📦 Volume inadapté ?** Ajustez la quantité proposée." %}

**💬 Messagerie Intégrée :** Utilisez le chat pour discuter avec l'annonceur, argumenter votre prix ou demander des précisions.
{% endcapture %}
{% include step.html index="8" title="Réception & Négociation" content=step8 %}

{% capture step9 %}
Une fois la campagne validée par les deux parties, on avance sur l'opérationnel :
- **Asile Colis :** Expédition des supports (respect du délai d'ingestion).
- **Digital :** Transmission des assets créatifs et intégration.
{% endcapture %}
{% include step.html index="9" color="green" title="Validation Opérationnelle" content=step9 %}

<hr class="my-6">

<h2 class="text-blue-200 mb-6">Phase 3 : Accélération & Autonomie</h2>

{% capture step10 %}
Ne dépendez pas uniquement des annonceurs externes. Soyez proactif avec vos propres fournisseurs !

{% include callout.html type="tip" content="**Le concept :** Invitez les marques que vous distribuez à acheter de la visibilité chez vous (Asile, Email, Social) via getinside.<br>• ✅ **Simplification :** Centralisez toute la gestion et la facturation au même endroit.<br>• ✅ **Rentabilité :** Générez des revenus additionnels sur des partenaires déjà acquis." %}
{% endcapture %}
{% include step.html index="🚀" color="blue" title="Trade Marketing (Vos Marques)" content=step10 %}

{% capture step11 %}
Vous connaissez d'autres marques ou e-commerçants ?
- Invitez-les à rejoindre la plateforme.
- Touchez une commission sur leur activité.
- [Voir le programme d'affiliation](affiliation).
{% endcapture %}
