# Étape 3 : Tracking, Validation et Diffusion

Une fois la maquette HTML prête, suivez ce processus rigoureux pour lancer la campagne via [app.getinside.media](https://app.getinside.media/).

## 1. Intégration du Tracking (Distributeur)
Avant de finaliser le HTML, le Distributeur doit se connecter à l'espace "Configuration et suivi" de la campagne pour récupérer deux éléments cruciaux :

1.  **Le Pixel de Tracking :**
    *   C'est une image invisible (1x1 px).
    *   **Action :** Copiez le code fourni et collez-le tout en haut ou tout en bas de votre code HTML (avant la balise `</body>`).
    *   *But :* Comptabiliser les ouvertures.

2.  **Les Liens Trackés (Redirections) :**
    *   **Action :** Remplacez tous les liens vers le site de l'Annonceur par les liens trackés fournis (format `gtinsi.de...`).
    *   *But :* Comptabiliser les clics et attribuer les ventes.

## 2. Review Technique (BAT Interne)
Le Distributeur envoie sa maquette finale (ou un lien miroir) à l'équipe getinside pour vérification technique.
*   **Email :** `benoit+news@getinside.fr`

getinside vérifie le tracking, le responsive et la conformité, puis génère un **lien de prévisualisation statique**.

## 3. Double Validation (BAT Final)
Ce lien de prévisualisation est envoyé aux deux parties (Annonceur et Distributeur).

*   **Vérification :** Chacun vérifie les textes, les visuels et le fonctionnement des liens.
*   **Action Requise :**
    1.  L'**Annonceur** se connecte sur [app.getinside.media](https://app.getinside.media/) et clique sur le bouton **"Valider la campagne"** (CTA).
    2.  Le **Distributeur** fait de même de son côté.

{: .important }
> La campagne est officiellement validée (Bon à Tirer) uniquement lorsque les deux parties ont cliqué sur le bouton de validation dans l'interface.

## 4. Diffusion et Confirmation
1.  Le Distributeur programme l'envoi sur son routeur (CRM).
2.  Une fois l'envoi effectué, le Distributeur retourne sur son espace "Configuration et suivi".
3.  Il clique sur **"Valider la diffusion"** pour clôturer l'opération et déclencher le suivi des stats finales.
