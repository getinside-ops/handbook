# Spécifications Techniques

Pour garantir un affichage optimal sur tous les clients mails (Outlook, Gmail, Apple Mail...), merci de respecter ces standards.

## Formats Standards
Les formats dépendent de l'emplacement négocié dans la newsletter du partenaire.

| Format | Dimensions (L x H) | Usage fréquent |
| :--- | :--- | :--- |
| **Bannière Large (Leaderboard)** | **600 x 150 px** (ou 640px) | Header ou Cœur de mail |
| **Pavé (Rectangle)** | **300 x 250 px** | Colonne latérale ou fin de mail |

## Contraintes Techniques
*   **Type de fichier :** JPG, PNG ou GIF.
*   **Poids max :** **150 Ko** (Recommandé < 80 Ko pour un chargement rapide sur mobile).
*   **Couleurs :** RVB.

### Bonnes pratiques
*   **Bordure :** Si votre bannière a un fond blanc, ajoutez une fine bordure grise de 1px (`#cccccc`) pour la délimiter du fond de l'email.
*   **GIF Animé :**
    *   Accepté, mais attention : Outlook (versions bureau) n'affiche que la première image du GIF.
    *   **Règle d'or :** L'information principale et le Call-to-Action doivent figurer sur la **première frame**.
*   **Texte :** Évitez les polices trop fines ou trop petites (minimum 16px pour être lu sur mobile).
