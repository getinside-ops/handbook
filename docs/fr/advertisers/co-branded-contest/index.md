# Jeu Concours Co-brandé

Le Jeu Concours est un dispositif puissant qui combine **notoriété** et **acquisition de leads (Data)**.

Le principe est simple : un e-commerçant partenaire héberge un jeu concours sur son site pour faire gagner vos produits à sa communauté. En échange de la participation, l'utilisateur accepte de recevoir vos offres.

## Les Bénéfices
1.  **Acquisition de Leads (Opt-in) :** Vous récupérez une base de données d'emails de prospects ayant consenti à recevoir vos communications.
2.  **Conversion massive :** Chaque participant (même perdant) reçoit un **code promotionnel** de "consolation" à la fin du jeu, générant des ventes immédiates.
3.  **Engagement :** La mécanique de jeu (Gamification) crée un lien fort avec la marque.

## Votre Investissement
*   **Dotation :** Vous fournissez et financez les lots principaux (ex: 3 x 100€ de produits).
*   **Récompense participant :** Un code promo générique (ex: -15%) pour tous les joueurs.
*   **Budget Média :** Coût de la campagne via getinside.
