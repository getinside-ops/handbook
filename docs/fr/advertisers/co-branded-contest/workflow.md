# Workflow du Jeu Concours

Ce dispositif nécessite une coordination précise pour la gestion des données personnelles (RGPD).

## 1. Définition de l'offre (Brief)
Vous devez fournir les éléments suivants via la plateforme :
*   **Les Lots (Gros lots) :** Ce qui attire les joueurs (ex: "Gagnez un an de produits").
*   **Le Code Promo (Lot de consolation) :** Ce qui convertit les joueurs (ex: "Merci d'avoir joué ! Voici -15% sur votre première commande").
*   **Les Visuels :** Packshots des lots et logo.

## 2. Mise en place (Côté Distributeur)
Le Distributeur crée une **Landing Page (Page dédiée)** sur son site e-commerce aux couleurs de l'opération.
Il active ses leviers de trafic (Newsletter, Push Web, Bannières site) pour attirer les joueurs.

## 3. Mécanique de collecte (Opt-in)
Lors de l'inscription au jeu, le participant coche une case (Opt-in Partenaire) acceptant que ses données vous soient transmises.

## 4. Transfert de Data & Activation
À la fin de la campagne (ou à fréquence définie) :
1.  Le Distributeur extrait la base des participants "Opt-in".
2.  Il vous transmet ce fichier de manière sécurisée (CSV).
3.  **Action requise de votre part :** Vous devez intégrer ces emails dans votre outil CRM (Mailchimp, Klaviyo, etc.) et leur envoyer un email contenant **le code promotionnel promis**.

{: .important }
> **Le Timing est clé :** Plus l'email contenant le code promo est envoyé tôt après la fin du jeu, meilleur sera le taux de conversion. Préparez votre template d'email à l'avance !

## 5. Tirage au sort
Vous effectuez le tirage au sort des gagnants des gros lots et gérez l'expédition des récompenses directement auprès d'eux.
