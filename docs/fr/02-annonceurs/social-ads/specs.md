# Spécifications Social Media
{: .fs-9 }

Pour maximiser l'engagement, vos contenus doivent respecter les codes du "Native Advertising".
{: .fs-6 .fw-300 }

<hr class="my-6">

<h2 class="text-blue-200 mb-4">📱 Vidéo (Reels & TikTok)</h2>
<p class="text-green-200 fw-700">✅ Format Recommandé pour la performance.</p>

<!-- Tableau en Markdown natif -->

| Critère | Spécification |
| :--- | :--- |
| **Ratio** | 9:16 (Vertical Plein Écran) - 1080 x 1920 px |
| **Durée** | 15 à 60 secondes idéalement. |
| **Son** | "Sound On" obligatoire (Musique libre de droit ou Voix off). |
| **Zone de sûreté** | Laissez une marge de **15% en haut et en bas**.<br><span class="fs-2 text-grey-dk-000">Ne placez pas de texte ou de logos importants dans ces zones (cachés par l'interface TikTok/Insta).</span> |

<br>

<h2 class="text-blue-200 mb-4">📷 Image (Feed Instagram)</h2>

<!-- Tableau en Markdown natif -->

| Format | Dimensions | Usage |
| :--- | :--- | :--- |
| **Vertical** | **4:5 (1080 x 1350 px)** | <em class="text-grey-dk-000">Recommandé car il occupe plus d'espace sur l'écran.</em> |
| **Carré** | 1:1 (1080 x 1080 px) | Format classique. |
| **Texte** | Max 20% de la surface | Privilégiez la légende (caption) pour le texte. |

<br>
