# Sponsored Social

Le Sponsored Social vous permet de monétiser votre audience sur les réseaux sociaux (Instagram, TikTok, Facebook) sans polluer votre ligne éditoriale.

## Le principe : "Partnership Ads"

Contrairement à l'influence classique où vous devez poster une publicité pour votre audience, ici nous utilisons la **Sponsorisation**.

1.  Vous postez un contenu qualitatif (en partenariat avec une marque).
2.  L'Annonceur investit son propre budget média pour diffuser **votre** post auprès de milliers de nouvelles personnes.

## Vos avantages
*   **Rémunération :** Vous êtes payé pour la création et/ou la publication.
*   **Croissance de Followers :** L'annonceur paie pour montrer VOTRE profil à une large audience. C'est de la publicité gratuite pour votre compte, vous permettant de recruter de nouveaux abonnés qualifiés.
*   **Qualité préservée :** Comme le post est sponsorisé (Dark post ou amplifié), il n'est pas nécessaire de saturer votre grille organique avec des messages commerciaux répétitifs.
