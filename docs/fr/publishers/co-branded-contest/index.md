# Jeu Concours Co-brandé

Le Jeu Concours est un format engageant qui vous permet d'animer votre communauté tout en monétisant votre audience.

## Pourquoi proposer ce format ?
1.  **Revenus Média :** L'Annonceur finance la campagne.
2.  **Animation Commerciale :** Vous proposez une expérience ludique et des cadeaux (financés par la marque) à vos clients, sans coût pour vous.
3.  **Qualification de Data :** C'est l'occasion de réactiver des clients inactifs ou de collecter de nouveaux opt-ins sur votre propre base.

## Le concept
Vous hébergez une page de jeu sur votre site (ou une page dédiée externe). Vous dirigez du trafic dessus via vos canaux habituels (Email, Push, Social).
