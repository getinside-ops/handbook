# Guide de mise en place

## 1. Création de la Landing Page (LP)
Vous êtes responsable de l'hébergement du jeu.
*   **Hébergement :** Sur votre domaine (ex: `mon-site.com/jeu-concours-marque`) pour rassurer le client.
*   **Design :** Co-brandé (Votre header habituel + l'univers de l'Annonceur).

## 2. Le Formulaire & RGPD (Point Critique)
Le formulaire d'inscription doit respecter strictement le RGPD pour permettre la monétisation de la donnée.

Il doit comporter impérativement la case à cocher suivante (non pré-cochée) :
> ☐ *J'accepte de recevoir les offres commerciales du partenaire [Nom de l'Annonceur].*

{: .warning }
Seuls les participants ayant coché cette case pourront être facturés/transmis à l'Annonceur. Les autres participent au tirage au sort mais ne sont pas monétisables en "Lead".

## 3. Plan d'activation
Pour garantir le succès (et donc le volume de leads), activez vos leviers puissants :
*   **Newsletter dédiée ou Encart :** "Tentez de gagner XXX€ chez [Marque]".
*   **Web Push Notification.**
*   **Bannières Home Page.**

## 4. Transfert des données
À la clôture du jeu :
1.  Exportez la liste des participants ayant validé l'opt-in partenaire.
2.  Transmettez le fichier (CSV/Excel) de manière sécurisée à l'Annonceur (ou via getinside).
3.  L'Annonceur se chargera d'envoyer le code promo de consolation. Vous n'avez pas à gérer l'envoi des emails transactionnels post-jeu.
